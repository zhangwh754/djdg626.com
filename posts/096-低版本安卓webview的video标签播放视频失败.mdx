---

## 使用 ffmpeg

### 查询视频的信息

```
./ffmpeg.exe -i video.mp4
```

### 降级视频的编码等级

```
./ffmpeg.exe -i ./response.mp4  -c:v libx264 -profile:v baseline output.mp4
```
