description: 页面使用Parent组件。Parent组件下使用多个Child组件。现在希望页面可以通过插槽传递一个组件/dom到Child组件中去，页面可以完全控制这个组件/dom的样式和事件
---

## 代码

```html
// page.wxml
<MParent>
  <button wx:for="{{5}}" wx:key="item" slot="test-{{item}}" data-index="{{index}}" bind:tap="onButtonClick">
    测试按钮{{item}}
  </button>
</MParent>
```

```html
// Parent.wxml
<MChild wx:for="{{5}}" wx:key="index" item="{{item}}">
  <view slot="c-test-{{item}}">
    <slot name="test-{{item}}"> </slot>
  </view>
</MChild>
```

```html
// Child.wxml
<slot name="c-test-{{item}}"></slot>
```
