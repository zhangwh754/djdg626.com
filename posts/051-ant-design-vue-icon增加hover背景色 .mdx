## 基本

```js
h(DeleteOutlined, {
  class: 'delete-icon',
  style: 'margin-left: 10px;cursor: pointer;',
  onClick: () => removeDateRange(index, type),
})
```

样式穿透+important

```css
:deep(.delete-icon:hover) {
  svg {
    fill: #2471a3 !important;
  }
}
```
